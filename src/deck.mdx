# Tipos de datos

---

export default Layout;

## String

Los strings son elementos que se componen principalmente de texto entre comillas( dobles, simples o [backticks](https://developer.mozilla.org/es/docs/Web/JavaScript/Referencia/template_strings)), también son llamados cadenas de texto.

**Ejemplo**: "Joss", 'ironhack', \`3\`

---

export default Layout;

## Number

Como su nombre lo dice, son un tipo de datos numérico. Nos permiten realizar operaciones lógicas y matemáticas.

**Ejemplo**: 1, 0, -23

---

export default Layout;

## Boolean

El tipo de dato booleano está compuesto solo de 2 posibles valores: **true** o **false**. Y también nos permiten realizar operaciones lógicas.

---

export default Layout;

## Undefined y Null

Existen 2 tipos de datos curiosos, ambos representan un valor negativo pero hay una ligera diferencia:

- **undefined** es un dato que no está definido en el programa o no existe.

- **null** es simplemente una asignación a proposito de 'algo' vacio.
